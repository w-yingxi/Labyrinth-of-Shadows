# Labyrinth-of-Shadows
====================

DESCRIPTION
===========
Labyrinth of Shadows is a maze puzzle game where the player must navigate through a maze to collect keys and escape. 
In the maze, the player needs to create a route to the destination by pushing movable blocks while avoiding randomly moving monsters.


RUNNING THE GAME
================
- Type 'python main.py' in terminal to run the game. 
- Please make the terminal tall to completely show the maze. 
- Do not change the terminal height after the game starts. 

CONTROLS 
========
- Move using W/A/S/D.
- Input R to restart the game and select a new maze randomly.
- Input Q to end the game. 
- Inputs are case-insensitive. 


GAMEPLAY INSTRUCTIONS
=====================
- You are the player😄. Your goal is to survive the monsters, collect all of the keys and escape through the exit.
- There are monsters 👾 which move in random directions. They will move after the player enters a valid movement command. 
- The player gets killed when stepping on a monster, or when a monster moves onto the same place as the player.
- The exit will only open once you collect all of the keys 🔑. It will turn from red🟥 to green🟩.
- You can push blocks ⬜ when nothing is behind it and you are next to it in the opposite direction. 
- You cannot walk through walls ⬛.
- 🟦 stands for one-way secret path. After you pass through it, it will turn into walls ⬛. Monsters 👾 cannot move onto it.
- There are 3 levels, with 1 monster, 1 key, 2 monsters, 2 keys, and 2 monsters, 3 keys respectively.
- Plan your route before moving and watch out for monsters!
